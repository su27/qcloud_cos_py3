Qcloud COS SDK for Python 3
===============================

version number: 0.1.0
author: Dan Su

Overview
--------

A 3rd-party SDK for Qcloud COS and Python 3

Installation / Usage
--------------------

To install use pip:

    $ pip install qcloud_cos_py3


Or clone the repo:

    $ git clone https://github.com/su27/qcloud_cos_py3.git
    $ python setup.py install
